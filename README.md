# Official Bihar Bowl® | From Bihar to the World

Welcome to the official repository of **Bihar Bowl**, India's fastest-growing Bihar-food chain. This application is a full-stack platform designed to manage our brand identity, franchise operations, and customer experience.

## 🚀 About Bihar Bowl
Bihar Bowl started its journey in 2020 with a vision to blend the health benefits of Sattu with the traditional "Litti Chokha" of Bihar. We are dedicated to preserving authentic wood-fired cooking methods while innovating for the global palate.

## ✨ Key Features
- **Authentic Menu:** Explore our wide range of traditional Litti, Sattu drinks, and Bihari delicacies.
- **Franchise Management:** Dedicated portal for franchise applications and operational documentation.
- **Store Locator:** Find our 50+ outlets across 20+ cities.
- **Admin Dashboard:** Secure management of menu items, reviews, and franchise applications.
- **Digital Menu:** QR-code based ordering system for branches.
- **Brand Identity:** Access to Mithila-themed design guides and brand standards.

## 🛠️ Tech Stack
- **Frontend:** React 19, Vite, Tailwind CSS (v4), Framer Motion, Lucide Icons.
- **Backend:** Node.js, Express.
- **Database:** SQLite (Better-SQLite3) for lightweight and fast data management.
- **Styling:** Modern utility-first CSS with a custom Mithila-inspired design system.

## 💻 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Mrsaurabhkumar123/bihar-bowl-web.git
   cd bihar-bowl-web
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```
   The app will be available at `http://localhost:3000`.

4. **Build for production:**
   ```bash
   npm run build
   npm start
   ```
